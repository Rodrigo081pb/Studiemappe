Seletor CSS

Seja mal-vindo

No CSS temos alguns seletores

  • Seletor por tipo: Busca elementos por uma tag HTML
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por classes(.): Busca elementos através do atributo "class"
  • Seletor universal(*) Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]) Selecione elementos que possuem um atributo
